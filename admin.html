📊 Google Sheets Workflow

Step 1 — Open Payments Sheet

Go to your Google Sheet → Payments tab. New submissions appear here automatically.

Open Google Sheet →

Step 2 — Verify Payment

Check UPI remark in your PhonePe/GPay transaction history. Match with student's submitted remark.

Format: PhysicsCourse_[StudentName]

Step 3 — Update Sheet

In Payments sheet, change the Verified column to YES for verified payments.

Step 4 — Grant Drive Access

Open Google Drive folder → Right-click → Share → Add the student's Gmail address with Viewer access.

Open Drive Folder →

Step 5 — Update Access Given

Change Access Given column to YES. Optionally send a confirmation WhatsApp.

⚙️ Apps Script Setup

1. Open your Google Sheet → Extensions → Apps Script

2. Paste the Apps Script code from apps-script.gs

3. Deploy → New deployment → Web App

4. Set "Who has access" to Anyone

5. Copy the deployment URL

6. Paste it in script.js → CONFIG.APPS_SCRIPT_URL

🎬 Add Reels

Add reels by populating the Reels_Data sheet with columns:

ID | Topic | Class | Tags | Difficulty | Video URL

Video URL format: https://www.youtube.com/embed/VIDEO_ID
